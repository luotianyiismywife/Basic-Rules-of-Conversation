# 泛化能力评测 — 用例清单 + 预期结果

> **使用方式**：将 `双人基础规则.md` 作为系统提示词（或附件），按以下场景依次输入对话，观察模型输出是否包含预期待办。

## 用例列表

### TC-GEN-01：餐厅点餐等上菜（有示例覆盖）

**前置上下文：** 完成阶段1初始化（模式选择→场景构建→情景设定完成）

**用户输入：**
```
今晚咱们去哪里吃饭呢？
```
（用户继续推进到餐厅点完餐）

**关键触发输入：**
```
嗯…咖喱蛋包饭必点的！再来一份唐扬鸡块和一杯蜜桃乌龙茶~darling你要不要也来杯喝的？
```

**预期行为：**
- ✅ 生成 `待办{事项: 吃上菜}` 或 `待办{事项: 吃饭}` 或等效
- 触发分类：d) 需要等待结果的动作（有示例）
